Built for fewer weight changes: each day flows power → heaviest lift → lighter/band work, so the dumbbell only changes 2–3 times and never jumps back up. On pull day all the dumbbell work is done first, then you set them down for the bands.
First 2–3 weeks: 2 sets only, leave reps in the tank, don't hit failure. Build to full sets once soreness fades. Intensity (close-ish to failure) matters more than piling on sets.
Picking a weight: last 2 reps should be hard but your form holds. When the top of the rep range feels easy, slow the tempo & add a pause before going heavier. (Weights shown are targets to grow into — start lighter.)
Weekly shape: Legs · Push · Pull · rest, once or twice through per week. Add 2–3 easy cardio sessions — 30–40 min where you can talk but not sing — on off days. Optional extra: a minute of farmer's carries (just walk holding the DBs) is great for posture.
